# ToDo List App

This project is a simple ToDo list application built using Django, JavaScript, HTML, and Bootstrap. Users can add, edit, and mark completed tasks.

![ToDo List Screenshot](screenshot.png)

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
